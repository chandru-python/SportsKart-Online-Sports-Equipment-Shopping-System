# 🚀 Project Overview

**SportsKart** is a modern Flask-based e-commerce web application designed for buying sports equipment online.  
The system provides a complete shopping experience with secure authentication, product browsing, cart management, order tracking, and an advanced admin dashboard.

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/chandru-python/SportsKart-Online-Sports-Equipment-Shopping-System.git
```

---

## 2️⃣ Move into Project Folder

```bash
cd SportsKart-Online-Sports-Equipment-Shopping-System
```

---

## 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Application

```bash
python app.py
```

---
